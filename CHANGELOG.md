# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Crypto Liquidation Tracker
- Support for 5 major cryptocurrencies (BTC, ETH, USDT, BNB, SOL)
- Multiple timeframe analysis (1h, 4h, 24h)
- Interactive bar chart visualization using Chart.js
- Dark mode interface with gradient effects
- Responsive design for mobile, tablet, and desktop
- Summary cards showing liquidation metrics
- Percentage difference calculation
- Error handling with user-friendly messages
- Mock data generation for demonstration
- Comprehensive documentation (README, ARCHITECTURE, CONTRIBUTING)

### Technical
- Modular JavaScript architecture
- CSS variables for theming
- ES6+ JavaScript features
- Chart.js 4.4.0 integration
- Semantic HTML5 markup
